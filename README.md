# 微信小程序研学自习室选座与门禁系统的实现与开发springboot

## 下载地址
https://bysj.bsitem.com/#/home/goodDetail?goodId=30233

## 技术介绍
微信小程序+Java语言+ssm框架/springboot框架（后端）+Mysql(数据库)+Vue前端框架（Java Web）

## 开发工具
微信开发者工具（小程序端）+IDEA或者MyEclipse(后台管理端)+Navicat(mysql数据库可视化工具)
